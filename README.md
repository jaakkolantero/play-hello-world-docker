# play-hello-world-docker

Simple helloworld scala app with play framework. Docker files included. 

## Getting Started

Run app and head to http://localhost:9000/

### Prerequisites

* scala ~2.12.6
* Play framework ~2.6.18
* SBT ~1.2.4


### Installing

to run app

```
sbt run
```

## Deployment

Deploy to now.js
``` 
./stageAndBuild
now
```
